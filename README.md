## Технологический стек

### Клиент (Frontend)
- **HTML** - разметка страниц
- **CSS** - стилизация страниц и адаптивная вёрстка
- **JavaScript** - клиентская часть и взаимодействие с серверном

### Сервер (Backend)
- **Node.js** - выполнения JavaScript на сервере
- **Express.js** - фреймворк работы с сервернной частью
- **Prisma** - ORM для работы с базой данных

### База данных
- **PostgreSQL** - реляционная СУБД

### Деплой (Развёртывание)
- **GitHub** - хранение кода и автоматический деплой при пуше
